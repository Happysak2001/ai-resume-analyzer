# AI Resume Analyzer

A simple AI-powered web app that analyzes resumes against job descriptions and provides insights.

## Features
- Resume vs Job Description matching
- Match score
- Missing skills and keywords
- Strengths and suggestions

## Tech Stack
- Python
- Streamlit
- Groq API (LLM)
